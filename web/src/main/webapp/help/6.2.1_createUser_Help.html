6.2.1 Create New User Help

Here you can create a new user account. Please fill in all fields. With the drop down menu label Active Study you can assign the user to a specific study. Underneath you can select a Role and the User Type.

You will also be prompted to enable Web Services for this user. This is important for only those users who will be using XML-based Web Services, together with the UI of ClinCapture.

You are also presented with two possibilities for the User Password. Either you can select the option to send the system generated User Password directly to the user via Email, or you choose to display the User Password to yourself as Admin and provide the user with the password later.

Please click the button to confirm the new user account or click the button to stop the process and to go back to the Administer User page.
