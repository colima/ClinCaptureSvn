6.2 Users Help

This section allows you to view all the users available to you. All users are ordered through their User Name. You can choose to reorder the list of users by toggling the ascending or descending arrow located next to the User Name header. Similarly, you can also reorder the user list using the user’s First Name, Last Name, or Status, and toggle their respective ascending or descending arrows.

Note: Under each First and Last Name are listed the user’s role within specific studies given in a “Study Name – Study Role” format.

With the Find function, located at the top of the User list, you can search for a set of users based on their User Name, First Name, or Last Name.

In addition to viewing all the Users, you are also presented with buttons to Create New User, to view the Audit User Activity and to edit the Lockout Configuration.

The Administer Users matrix displays the following columns:

1. User Name This cell indicates the name of the user.
2. First Name This cell indicates the first name of each user.
3. Last Name This cell indicates the last name of each user.
4. Status This cell indicates the status of each user.
5. Actions (Global Record) View

Clicking on the View icon will allow you to see all the details associated with the particular user.

Edit

Clicking on the Edit icon will allow you to edit the user’s First Name, Last Name, Email, Institutional Affiliation, and User Type. From here, you can also reset the Users password.

Set Role

Clicking on the Set Role icon will allow you to assign a Role within a Study for the particular user.

Remove

Clicking on the Remove icon will allow you to remove the particular user. From this point on, the user will no longer be associated with any study and any roles, unless the user is restored. Note that separate Roles can also be Removed, Restored and Deleted. If you Remove a User, however, you will not have access to modify those Roles.

Restore

Clicking on the Restore icon will allow you to restore the particular user. From this point on, the user will have access to the assigned studies under specific roles, unless that user or Role is removed.
6. Actions (Study/Site Level) Edit

Clicking on the Edit icon will allow you to edit the user’s Role at that particular Study or Site.

Remove

Clicking on the Remove icon will allow you to remove the particular user Role. From this point on, the user will no longer have access to that particular Study or Site.

Delete

Clicking on the Delete icon will allow you to completely delete that user Role. You can always reassign the user to that Role later.
