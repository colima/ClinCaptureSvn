6.4.2 Scheduled Import Jobs

Here you can view all your existing jobs to import data. Below the list you will find a button entitled Create a new Import Data Job. This link will direct you to the 'Create Scheduled Job: Import Dataset' page, where you can configure a new job to import data.
Please fill out the fields named Import Job Name and Import Job Description. The default path to import data is displayed. If you want to provide another path, please enter it in the field next to it. After this, please define the start date and time as well as the frequency the job should be executed.
Please provide an Email Contact address in the last field.

The Find function, located at the top of the job list, allows you to search for a specific job based on the Name, Previous Fire Time, Next Fire Time, Description, or Study.

The View Scheduled Import Data Jobs matrix consists of the following columns:

Name: This cell indicates the Name of the job.
Previous Fire Time: This cell displays the last date and time the job was done.
Next Fire Time: This cell displays the next time the job will be executed.
Description: The description you typed in during the process of creation of the job, will be listed in this cell.
Study: Indicates the study (or site) the job belongs to. Note that XML assigned to another Study or Site will not import correctly.
Action: View

Clicking on the View icon will bring you to the View Import Job page to see all the details associated with this job.

Edit

Clicking on the Edit icon will bring you to the Update Scheduled Job page where you can edit the job details.

Remove

To remove a job means to pause it by clicking on the Remove icon. The job will remain in the database and can still be restored and modified or deleted at any time. It will not run until it is restored.

Restore

If a subject has a status of inactive, which means that it was removed, you can restore it to the study by clicking on the Restore icon.

Delete

If you finally want to delete a job you can click on the Delete icon and the job will be wiped from the system. Keep in mind that this delete cannot be undone.
