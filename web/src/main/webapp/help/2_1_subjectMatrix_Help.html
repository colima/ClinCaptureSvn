2.1 View All Subjects Help

This section allows you to view all the subjects that are enrolled in your current study. You can choose to reorder the list of subjects by clicking on a column title. The ascending or descending arrow located next to the column header indicates the sorting.

In the top of the table you can use the green arrows to navigate through the pages. If you have less than 15 subjects in your study the arrows are greyed out, which means that there is only one page to display.
With the drop-down menu next to the arrows you can choose how many subjects should be displayed on one page (15, 25, or 50).

With the Show More link you can expand the subject matrix to include additional columns containing Subject Status, OID, Sex, Secondary Label and Group Assignment:

Subject Status: This cell indicates the Subject Status. It could be available, signed, removed, or auto-removed.
OID: OIDs are a system generated identifier which are used for the importing of data, creation of rules and in CDISC ODM exporting of data.
Sex: This cell indicates the sex of each subject. Note that it might be a different text, set in the Study Parameters section.
Secondary Label: This cell indicates the Secondary Label of each subject. Note that it might be a different text, set in the Study Parameters section.
Group Assignment: A Subject Group Class is a class of group categories that are related and mutually exclusive. Once Subject Group Classes have been created, subjects may be assigned to one of the groups. The group class to which a subject has been assigned will be displayed in this column.

To collapse the table click the Hide link, you will see instead of the Show More link, in the top of the table if it is expanded.

With the drop-down menu, Select An Event in the top of the subject list, you can filter the subjects by an event. You can also filter by entering free text or by selecting a pre-defined choice from the drop-down menu, in the grey boxes underneath each column title. To apply the filter select the Apply Filter link.
To reset the filter click on the Clear Filter link next to it.

The subject matrix provides you with the following columns by default:

1. Study Subject ID This cell provides you with the Study Subject ID of each individual subject.
2. Study Events For each subject, you will see the Study Events with their status icon and next to it the number of occurrences if it is a Repeating Event. If you click of one of these icons you can schedule the event, view or enter data as well as edit or remove the event from the subject. You can also see Discrepancy Notes for the Subject, Event, or individual CRF.
3. Actions (depending on role) View

Clicking on the View icon will bring you to the View Subject page to see all the details associated with the particular subject.

Remove

To remove a Study Subject you can click on the particular Remove icon.

Restore

If a subject has a status of inactive, which means that it was removed, you can restore it to the study by clicking on the Restore icon.

Reassign

If you want to reassign the Study Subject to another study or site you can click on the Reassign icon and you will be brought to the 'Reassign Study Subject' page.

Sign

If you want to sign the Study Subject you can click on the Sign icon and the Subjects Event Status changes to signed. Note this option is typically only open to Investigators.

Perform SDV

If you want to perform Source Data Verification on the Study Subject you can click on the SDV icon and be taken to the SDV page for that Study Subject.

Lock

If you want to lock the Study Subject you can click on the Lock icon and the Subjects Event Status changes to locked.

Calendared Events

If your Study Subject has any Calendared Events scheduled or assigned, the Calendared Event icon will appear. Note that if there is a Event which is out of scope for that Subject, it will be changed to a red icon:

Discrepancies

At this part of the Subject Matrix you can also look at the Notes and Discrepancies for an entire Subject. Note that this flag changes to yellow: when all Notes for the Subject are set to Updated. If all notes are Closed or do not exist, the flag will not show.
