4.7 Edit Dataset Help

This section allows you to edit an existing dataset for your current study. The user interface presents you with a visual model as well as written instructions detailing the five steps that need to be taken in order to complete the entire process.


Step 1: Select Study Events

In the task pane on the left side you can select the Study Events and their related CRFs that you would like to include or remove from your dataset. You can select the specific items you want to include/exclude from your dataset by checking the appropriate item box. If you want to add or remove more items from different CRFs to your dataset, go on with the button otherwise click the button to save your changes and to define the temporal scope.


Step 2: Define Temporal Scope

If you want, you can specify the study’s temporal scope here. If you specify a month, you must also specify the year.
Go on by clicking the button.


Step 3: Specify Dataset Properties

Here you can specify properties for your dataset. Changing the name of the dataset will create a copy.
If you choose to output your data in CDISC ODM XML format you can specify additional information about MetaDataVersion ODM ID, MetaDataVersion Name, Previous Study ODM ID and/or Previous MetaDataVersion ODM ID.
Upon completion, please click the button to continue.


Step 4: Confirm Dataset Properties

Here you can click the button to go to the next page, where you can choose your export format or the button to change the properties.


Step 5: Download Data

Here you can export your dataset using one of the available formats. Once the file is generated, it can be saved locally to your hard drive and viewed in a text editor, an excel spreadsheet, or any of a number of analytical software applications. Data may also be viewed online in HTML format.

See the Export Help Page for special information about downloading files over SSL connections using Internet Explorer.
