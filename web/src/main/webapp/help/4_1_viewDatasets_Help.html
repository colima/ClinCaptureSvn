4.1 View Datasets Help

This section allows you to view all the datasets that are assigned to your current study. All datasets are ordered using their Dataset Name's. You can choose to reorder the list of datasets by toggling the ascending or descending arrow located next to the Dataset Name header. Similarly, you can also reorder the subject list using the dataset’s Description, Created By, Created Date, or Status, and toggle their respective ascending or descending arrows.

The Find function, located at the top of the dataset list, allows you to search for a set of datasets based on their Dataset Name, Description, Created By or Created Date.

In addition to viewing all the datasets, you are also presented with links to Show only My Datasets and to Create Dataset.

1. Dataset Name This cell indicates the Name of each Dataset.
2. Description This cell provides a brief description for each dataset.
3. Created By This cell indicates the creator of each dataset.
4. Created Date This cell indicates the date the dataset was created.
5. Status This cell indicates the current status of each dataset.
6. Actions (depending on role) View

Clicking on the View icon will allow you to see all the details associated with the particular dataset on the View Dataset page.

Edit

Clicking on the Edit icon will bring you to the Edit Dataset page where you can add and/or remove items from your dataset.

Remove

Clicking on the Remove icon will allow you to remove the selected dataset. The dataset will no longer be accessible to users of ClinCapture. Once removed, the dataset can only be restored by a user with an appropriate role.
Note: Removing a dataset will not delete any data from the study.

Export Dataset

Clicking on the Export Dataset icon will allow you to export the dataset using one of the available formats. Once the file is generated, it can be saved locally to your hard drive and viewed in a text editor, an excel spreadsheet, or any of a number of analytical software applications. Data may also be viewed online in HTML format.

Restore

Clicking on the Restore icon will allow you to restore the dataset to your study.
