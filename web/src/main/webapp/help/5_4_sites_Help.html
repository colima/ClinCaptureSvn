5.4 Sites Help

This section allows you to view all the sites that are a part of your current study. All sites are ordered through their Name. You can choose to reorder the list of sites by toggling the ascending or descending arrow located next to the Name header. Similarly, you can also reorder the subject list using the site’s Unique Identifier, Principal Investigator, Facility Name, or Date Created, and toggle their respective ascending or descending arrows.

The Find function, located at the top of the site list, allows you to search for a set of sites based on their Name, Unique Identifier, Principal Investigator, or Facility Name.

1. Name This cell indicates the name of each site.
2. Unique Identifier This cell indicates the Unique Identifier of each site.
3. OID This cell indicates the Object Identifier (OID) of each site.
4. Principal Investigator This cell indicates the name of the principal investigator of each site.
5. Facility Name This cell indicates the name of the facility of each site.
6. Date Created This cell indicates the date the site was created.
7. Status This cell indicates the status of each site. It could be available, removed, or auto-removed.
8. Actions View

Clicking on the View icon will allow you to see all the details associated with the particular site.

Edit

Clicking on the Edit icon will allow you to edit all the properties of the site, including the Name, Unique Protocol ID, Secondary IDs, Principal Investigator, Brief Summary, Start Date, Estimated Completion Date, Facility (Name, City, State, ZIP, Country, Contact Name, Contact Degree, Contact Phone, Contact Email), and Study System Status.

Remove

Clicking on the Remove icon will allow you to remove the particular site. From this point on, only the Administrator can access and reverse this change if necessary.

Restore

Clicking on the Restore icon will allow you to restore the particular site. From this point on, the site will be assigned back to the specific study, unless the site is removed.

In addition to viewing all the sites, you are also presented with a link to Create a New Site in the top of the table.
