6.3 CRFs Help

This section allows you to view all the Case Report Forms (CRFs) that are available for use in your studies and sites. All CRFs are ordered through their CRF Name. You can choose to reorder the list of CRFs by toggling the ascending or descending arrow located next to the CRF Name header. Similarly, you can also reorder the CRF list using the CRF’s Date Updated, Last Updated By, Version OID, or Date Created, and toggle their respective ascending or descending arrows.

With the Find function, located at the top of the CRF list, you can search for a set of CRFs based on their CRF Name.

In addition to viewing all the CRFs, you are also presented with links to download a Blank CRF Template, or a Blank Randomization CRF Template as well as a button to Create a New CRF.

The Administer Case Report Forms (CRF) matrix displays the following columns:

1. CRF Name This cell indicates the Name of each CRF.
2. Date Updated This cell indicates the date the CRF was last updated.
3. Last Updated By This cell indicates the user who last updated the CRF.
4. CRF OID This cell indicates the Object Identifier (OID) of each CRF.
5. Versions This cell indicates the specific version of a CRF.

6. Date Created This cell indicates the date the CRF was created.
7. Owner This cell indicates the owner who originally created the CRF.
8. Status This cell indicates the status of each CRF. It can be available, locked, removed, or auto-removed.
9. Download A Download Spreadsheet icon in this cell indicates that there is a downloadable Excel file of this CRF version. You can use this to create new versions of a CRF.
If you have problems wih downloading a file, please click here.
CRF Versions which have no downloadable file do not have an icon and are instead indicated with “N/A”.
10. Actions View

Clicking on the View icon will bring you to the View CRF Details page.

Edit

Clicking on the Edit icon will allow you to edit the CRF’s Name and Description.

Archive

Clicking on the Archive icon will allow you to archive the CRF.

Un-Archive

Clicking on the Un-Archive icon will allow you to un-archive the CRF.

Remove

Clicking on the Remove icon will allow you to remove the particular CRF. From this point on, only Administrators can access and reverse this change if necessary.

Restore

Clicking on the Restore icon will allow you to restore the particular CRF. From this point on, the CRF will be available for studies and data entry, unless it is removed again.

Delete

Clicking on the Delete icon will delete this CRF version. It can not be restored.

Create New Version (only available for the original CRF)

Clicking on the Create New Version icon will allow you to add a new updated excel spreadsheet. Doing so will still keep the “(original)” and other versions intact.

Browser Settings for Downloading Files

There are two important settings you should know about to make sure you are able to download files from ClinCapture:

1. Popups: ClinCapture uses popups to provide information about a data file before you save it to your disk. When you click on a link to download a data file, please be sure you have any popup blockers turned off.

2. Internet Explorer Bug: ClinCapture is often configured to work over a secure SSL internet connection (this is the recommended configuration). If your installation of ClinCapture is set up this way, you will see a ‘lock’ icon on the browser and the URL will start with "https://".

Due to a bug in Internet Explorer, some users of that web browser will have difficulty downloading files over SSL. To fix this, you must change your browser Internet Options. Take the following steps:

  1. Go to ‘Tools -> Internet Options’
  2. Click on the ‘Advanced’ tab at the top of the dialog box.
  3. Scroll down to the section called ‘Security’.
  4. Find the line entitled ‘Do not save encrypted pages to disk’.
  5. Check the box on this line.
  6. Click ‘OK’

You should now be able to download files from ClinCapture.
