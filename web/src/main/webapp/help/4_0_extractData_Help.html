4. Extract Data Help

This section is designed to keep track of all the datasets in your current study, view datasets and/or create a new dataset.

All datasets are ordered by their Dataset Name. You can choose to reorder the list of datasets by toggling the ascending or descending arrow located next to the Dataset Name.

Similarly, you can also reorder the list using the Description, Created By, Created Date, or Status and toggle their respective ascending or descending arrows.

The Find function, located at the top of the datasets list, allows you to search for datasets based on their Dataset Name, Description, Created By, or Created Date.

Possible Actions you can perform, depending on your role, are:

View

Clicking on the View icon will bring you to the View Dataset page to see all the details of the dataset.

Edit

To Edit an existing Dataset you can click on the particular Edit icon. This will bring you to the Edit Dataset page.

Remove

To remove a dataset you can click on the Remove icon.

Export Dataset

If you want export the dataset you can click on the Export Dataset icon and you will be directed to the Download Data page.

In addition to viewing the latest datasets, you are presented with a button to Create Dataset as well as a link at the top of the page to View My Datasets.
