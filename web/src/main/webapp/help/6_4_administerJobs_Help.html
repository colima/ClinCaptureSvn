6.4 Jobs Help

This section is designed to keep track of all jobs. Jobs are tasks that run independently of a users' work on ClinCapture. You can configure and administer your import and export jobs to run at any time of day, and run repeatedly over time. This is important for generating weekly reports, for example, or importing remotely from another data source.

You are provided with two links to View All Export Data Jobs and to View All Import Data Jobs.
