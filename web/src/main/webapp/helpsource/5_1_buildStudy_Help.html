5.1 Build Study Help

The Build Study page is designed to help you build and configure your study by following a task based approach. The tasks are divided into 7 steps: Create Study, Create CRF, Create Event Definitions, Create Subject Group Classes, Create Rules, Create Sites, and Assing Users.

In the top of the table you can choose the study status from the drop-down menu. It could be available, locked, or frozen.

The displayed table provides you with additional information about the process of the creation of your current study.

1. Task This cell indicates the name of each task. During the whole process there are 7 tasks. Step 6 and 7 will be displayed after completion of step 1 to 5.
2. Status This cell indicates the status of each task. It could be Not startet, In Progess, or Completed.
3. #'s of This cell indicates the number of occurence in your current study, e.g. 10 CRFs. In case of task 1 (Create Study) it displays n/a.
4. Mark Complete If you have finished each step you can check the Mark Complete box and click the button.
