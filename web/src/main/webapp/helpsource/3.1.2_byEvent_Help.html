3.1.2 View By Event CRF Help

The table 'View By Event CRFS' shows the following columns by default:

1. SDV Status Displays a checkbox or a SDV Status icon if SDV was performed for this subject.
2. Study Subject Id Displays the Id of the Study Subject.
3. Site Id This cell indicates the Site ID the event is assigned to.
4. Event Name This cell presents the name of the event with the occurrence number in parenthesis.
5. Event Date Displays the day the event was performed.
6. CRF Name / Version This cell indicates detailed information about the name and version of the CRF.
7. SDV Requirement Shows the type of SDV that is required.
8. CRF Status This cell displays the status icon of the event CRF. It could be completed or locked.
9. Actions If you want to set SDV for an event CRF you need to click on the button.

In the top of the table you can use the green arrows to navigate through the pages. If you have less than 15 subjects in your study the arrows are greyed out, which means that there is only one page to display.
With the drop-down menu next to the arrows you can choose how many subjects should be displayed on one page (15, 25, or 50).

With the Show More link you can expand the subject matrix to include additional columns containing Person Id, Secondary Id, Enrollment Date, Subject Status, Last Updated Date, Last Updated By and Study Event Status:

Person Id This cell indicates the specific ID of the person.
Secondary Id To display the secondary Id of the subject.
Enrollment Date This cell indicates the specific ID of the person.
Subject Status This cell includes the enrollment date of the subject in the study.
Last Updated Date It displays the date the event was updated the last time.
Last Updated By Provides you with the name of the user who performed the last update of the event.
Study Event Status This cell indicates the status of the specific study event. It could be available, signed, removed, or auto-removed.

To collapse the table click the Hide link, you will see instead of the Show More link, in the top of the table if it is expanded.

You can filter by entering free text or by selecting a pre-defined choice from the drop-down menu, in the grey boxes underneath each table header. To apply the filter select the Apply Filter link.
To reset the filter click on the Clear Filter link next to it.

Source Data Verification

You can choose one or more Study Subject Id's to perform SDV for the Event CRFs. If you want to select all listed Study Subject Id's you can also hit the All Shown link in the top of the table. To uncheck click the None link next to it.
To set SDV click on the button below the table.

You can also remove Source Data Verification by clicking on the SDV icon under SDV Status.
