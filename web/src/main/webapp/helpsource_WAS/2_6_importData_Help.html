2.6 Import Data Help

This section allows you to import CRF data for particular subjects.

To import data, the follow criteria must be met:

  1. The Study must already exist within the particular instance of OpenClinica
  2. The CRFs and Event Definitions must be created
  3. The study Subject must be created
  4. The study event must be scheduled for the particular subject.

Once these criteria have been met, data can be imported into the CRFs via XML import file. Therefore you can click the button to serach for the XML file on your computer. Click Open to upload the file and the button to go on. Also you can cancel the action by clicking the button.

For more information on Importing of Data, please click here.
