2.2 Add Subject Help

This section allows you to enroll a new subject for your current study or site. You are presented with a graphical model as well as written instructions detailing the two steps necessary to enroll a subject. The latter step is subdivided into four individual steps in order to schedule a new study event for your newly enrolled subject.


Step 1: Enroll Subject

Here you can enter the Subject information. At minimum, please fill all requied fields. To enroll the subject and to add a study event for your subject, select the the button and go on with Step 2.

You can also choose to click the button if you wish to enroll other subjects at this time.

Or you can choose to click the button if you would like to add a study event for your subject at a later time. You will be brought to the 'View Subject' page.


Step 2: Add New Study Event

Here you can schedule a new study event for your newly enrolled subject. At minimum, please fill in the fields titled Study Event Definition, Location, and Start Date/Time. Upon completion, please click the button to continue.
You will be directed to the 'Enter or Validate Data for CRF' page.


Step 2 (a): Enter or Validate Data

Here you are presented with a Study Event overview. This section allows you to view all the CRFs included in your particular Study Event.

1. CRF Name This cell indicates the name of each CRF.
2. Version This cell indicates the version of each CRF.
3. Status This cell indicates the current status of each CRF.
4. Initial Data Entry This cell indicates the initial data entry for each CRF.
5. Double Data Entry This cell indicates the double data entry for each CRF.
6. Actions (depending on role) Enter Data

Clicking on the Enter Data icon will allow you to enter data into the particular CRF. It also allows you to continue with the creation of a new Study Event.

View Default

Clicking on the View Default icon will allow you to see the particular CRF. Here you can not insert data into the CRF.

Print

To print a blank version of the CRF click the Print icon. You can use this e.g. for you sourca data documentation.

Remove

To remove an Event CRF from the subject click the Remove icon.

Delete

If you want to delete a particulary Event CRF click on the Delete icon.

You can also update the Study Event Data by clicking on the Edit Study Event link or icon in the top of the first table.


Step 2 (b): Enter Data - CRF Info

If you click on one of the Enter Data (Paper & Pencil Icon) icon under Action of an event CRF you will be directed to the 'Initial Data Entry' page. Here you can find CRF Infos in the top section and underneath the CRF itself.

Note: Please fill in the fields titled Interviewer Name and Interview Date (if required) before proceeding to enter data for each section of the subject’s Event CRF.


Step 2 (c): Enter Data - Event CRF sections

Here you can enter data for the items in the Event CRF.

Note: It is not necessary to complete the Event CRF in one sitting. You can click the button on the top or bottom of the page to save the provided entries and to return to it at a later time. You will be directed back to the 'Enter or Validate Data' page. If you click the button you will be directed to the 'View all Subjects' page.


Step 2 (d): Mark Event CRF Complete

On the last section of each Event CRF you will find the Mark CRF Complete box to finalize the data entry process for your specific Event CRF. Only after you have completed the Data Entry process in Step 2(c), you should check the Mark CRF Complete box. If a password is required, you have to enter your user name and password to mark this CRF as completed.
