2.1.1 View Subject Help

You can minimize and maximize each section within the 'View Subject' page. The sections available to view are Subjects Record, Events, Group, and Global Subject Record. You can also view the audit logs by clicking on the Audit Logs link in the top.

Subjects Record

This matrix provides you with general information about the Subject, e.g. Study Subject ID, Study name, or Unique Protocol ID. Other fields depend on the settings of your study, e.g. Date of birth or Year of birth.

Back to Top

Events

In the Event Section you can see a list of all events assigned to this subject. All Events are ordered by their Start Date, with the latest on top. You can choose to reorder the list of events by toggling the ascending or descending arrow located next to the Start Date.

Similarly, you can also reorder the event list using the event name, Location, or Status, and toggle their respective ascending or descending arrows.

The Find function, located at the top of the event list, allows you to search for a set of events based on their Event name or the Location.

In addition to viewing all the Events, you are also presented with a link to Add New Event, which will direct you to the Schedule Event page.

The Event matrix provides you with the following columns:

Event (Occurrence Number): This cell indicates the name of the Event and in parenthesis the number of occurrence.
Start Date: The column Start Date provides you with the start date of each event.
Location: This cell indicates the location of the event.
Status: This cell presents the status of the event. It can be scheduled, data entry started, completed, and signed.
Not scheduled events will not be displayed in this matrix.
Actions: (depending on role) View

Clicking on the View icon will direct you to the Enter or Validate Data for CRFs page.

Edit

If you want to edit the Event you can click on the Edit icon and you will be directed to the Update Study Event page.

Remove

To remove an event from the Subject Record you can click on the particular Remove icon. The event and all data associated with it in this Study/Site will be removed. It may be restored by a system administrator.

Restore

If an Event was removed a system adminitrator can restore it to the study by clicking on the Restore icon. The event and all data associated with it in this Study/Site will be restored.
CRFs: This cell provides you with information about the Name of the CRF, the Version, the Status, the Updated date and the available Actions. The actions are similar to the actions on the Enter or Validate Data page (Step 2 (a)).

Group

You can expand the Group matrix under the event matrix by clicking on the icon. You will then see a list of all containing groups in this study with information about the Subject Group Class name, the Study Group as well as some Notes.
Additionally you are provided with a link to Assign Subject to Group.

Global Subject Record

Also, by expanding the Global Subject Record matrix you can see general information about the subject. In the top of the table you are provided with a link to Edit Record.

Audit Logs

The Audit Logs link, above all matrixes, will open in a popup window and displayes all captured transactions concerning this particular subject.
